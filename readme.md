# Biological Learning

## Installation

### 1. Create and Activate a Virtual Environment

On macOS and Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

On Windows:
```powershell
python -m venv venv
venv\Scripts\activate
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Install the Package Locally

```bash
pip install -e .
```
